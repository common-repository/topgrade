== Description ==

Topgrade allows you to quickly and easily add fun and engaging quizzes to your Wordpress website for free. 
Quizzes are a great way to increase user engagment on your site. 
To see the types of quiz questions you can add, see our example quiz - https://topgradeapp.com/playQuiz/world-geography

== Installation ==

1. Add the `topgradeapp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click Topgrade Admin in the menu bar and follow the instructions to add a quiz.

== Frequently Asked Questions ==

= How many quizzes can I make? =

You can make an unlimited number of quizzes/flashcards.

= Which pages of my Wordpress site can I add quizzes to? =

You can add quizzes to any page on which you're able to add HTML.

= How can I add a quiz? =

Please activate the Topgrade plugin and click the 'Topgrade Admin' menu button to access the Topgrade Admin page which explains how to quickly add a quiz in 3 simple steps.
